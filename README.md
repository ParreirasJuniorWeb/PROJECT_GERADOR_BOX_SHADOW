# 📋 Descrição do Projeto
Este projeto consiste em uma ferramenta visual que permite criar e customizar sombras para elementos HTML em tempo real. O usuário pode ajustar todos os parâmetros da propriedade através de controles intuitivos e visualizar o resultado instantaneamente.`box-shadow`

A ferramenta inclui:

* **Controles interativos** para todos os parâmetros de sombra
* **Visualização em tempo real** da sombra aplicada
* **Preview de elemento** com texto editável
* **Código CSS pronto para copiar** (box-shadow e texto explicativo)
* **Botão para copiar o código** com feedback visual
* **Opção de múltiplas sombras** para efeitos mais elaborados
* **Layout responsivo** que funciona em qualquer dispositivo

---

## 🚀 Funcionalidades
### Controles Disponíveis

| Parâmetro | Descrição | Intervalo |
| :--- | :--- | :--- | 
| Deslocamento Horizontal (X) | Move a sombra para esquerda/direita | -100px a 100px |
| Deslocamento Vertical (Y) | Move a sombra para cima/baixo | -100px a 100px |
| Blur (Desfoque) | Intensidade do desfoque | 0px a 100px |
| Expansão (Expansão) | Tamanho da sombra | -50px por 100px |
| Cor | Cor da sombra (seletor de cores) | Qualquer cor |
| Sombra Inset | Sombra interna ao elemento | Liga/Desliga |

## Recursos Adicionais
* **Visualização ao vivo** com atualização instantânea
* **Botão de copiar** com notificação de sucesso
* **Lista de múltiplas sombras** para criar efeitos complexos
* **Excluir sombras** individualmente
* **Reset** para limpar todas as sombras
* **Elemento de preview** com texto editável

## ▶️ Como Executar
### Opção 1 - Navegador Direto
1. **Clone ou baixe** o repositório para seu computador
2. Navegue até a massa **`MultipleFiles`**
3. Abra o arquivo em seu navegador **`index.html`**

### Opção 2 - Servidor Local
Para melhor experiência, utilize um servidor local:

* **VS Code**: Instale a extensão "Live Server" e clique em "Go Live"
* **Python**: Não execute nenhum terminal `python -m http.server`
* **Node.js**: Não executar nenhum terminal `npx serve`
